## Role in VAPT
Burp Suite is used for manual web application security testing focused on OWASP Top 10 vulnerabilities.


## Skills Demonstrated
- Web Application Security Testing (OWASP Top 10)
- Vulnerability Validation & False Positive Reduction
- Proof-of-Concept (PoC) Development
- Security Misconfiguration Identification


## Testing Scope
- OWASP BWA
- DVWA
- Intentionally vulnerable web services


## Key Activities

### Intercepting & Modifying Requests
Used to:
- Test input validation
- Identify insecure parameters
- Analyze authentication flows

### Manual Vulnerability Testing
Focused on:
- SQL Injection
- XSS
- Authentication & session issues
- Security headers misconfiguration

### Validation
Burp is used to confirm scanner findings and eliminate false positives.


## Example Workflow
1. Proxy traffic through Burp
2. Identify suspicious parameters
3. Manually manipulate requests
4. Observe server-side behavior
5. Capture evidence for reporting


## Professional Note
Burp Community is intentionally used to demonstrate strong manual testing skills without reliance on automation.
