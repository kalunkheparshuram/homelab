# Cybersecurity Homelab Overview

This project documents a personal cybersecurity homelab built using VirtualBox for hands-on learning in:
- Network Security
- Web Application Security
- Active Directory
- Vulnerability Assessment
- Penetration Testing
- Lab Components
- Machine	Purpose
- Kali Linux	Attacker Machine
- Windows 10 Enterprise	Client Machine
- Windows Server 2019	Domain Controller
- Metasploitable 2	Vulnerable Linux Machine
- OWASP BWA	Vulnerable Web Applications

## Skills Demonstrated
- Virtualization
- Windows Administration
- Linux Administration
- Active Directory
- Network Enumeration
- Web Security Testing
- Vulnerability Assessment
