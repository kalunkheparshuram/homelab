# Segmented VAPT Homelab – VirtualBox

This repository documents a self-designed **Vulnerability Assessment and Penetration Testing (VAPT) homelab** built using **Oracle VirtualBox**.  
The lab simulates a **segmented enterprise network** to demonstrate realistic internal security assessment workflows, controlled exploitation, and professional reporting.
The project focuses on **methodology, validation, and documentation** rather than tool dumping.

## Objectives

- Design a multi-zone virtual network simulating real enterprise environments
- Perform structured internal VAPT assessments
- Validate vulnerabilities and reduce false positives
- Demonstrate professional-grade documentation and reporting
- Build a GitHub portfolio aligned with junior VAPT / SOC analyst roles
